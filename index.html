🎓 Studienmanager

Willkommen bei deinem persönlichen Studienmanager!

Verwalte und analysiere dein Studium direkt, sicher und ohne Server über GitHub Pages.

Was kann diese Website?

Mit dieser Web-App kannst du dein offizielles Notenblatt analysieren lassen und visuelle Einblicke in deinen Studienfortschritt gewinnen. Es rechnet dir automatisch deine gewichtete Gesamtnote (NC) sowie deine ECTS-Fortschritte aus.

🛡️

Datenschutz-Garantie

Da diese App komplett mit lokalem JavaScript in deinem Browser läuft, werden deine Noten und Dokumente niemals auf einen fremden Server hochgeladen. Alles bleibt absolut privat.
